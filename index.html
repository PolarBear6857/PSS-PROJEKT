KÓD PRO ARDUINO

        
            int led = 6;
            int sensor = A1;

            void setup() {
                Serial.begin(9600);
                Serial.println("test");
                pinMode(led,OUTPUT);
                pinMode(sensor, INPUT);
            }

            void loop() {
                int light = analogRead(sensor) / 4;
                analogWrite(led,255 - light);
                Serial.println(light);
                delay(200);
            }

PRINCIP FUNGOVÁNÍ

Tento projekt má za cíl vytvořit čítač osob na bázi fototranzistoru.

Konkrétně používá princip jednocestné závory, která zjišťuje přerušení paprsku světla mezi emitorem (LED) a senzorem (Fototranzistor).

Arduino při každém přerušení světelné závory přičte jednoho člověka do celkového počtu osob, které závorou prošli.


Plakát ke stažení zde: (POZOR! není vhodné pro mobilní zařízení. Plakát je formátu A0)
